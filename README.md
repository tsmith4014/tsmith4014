## Selected project

**ABHES Q&A bot** — Parsed a 225-page regulatory PDF into an 800+ entry Q&A lookup using GPT-4 and multiple GPT-3.5 runs (~15 minutes, sub-$4 compute) to replace a slow, expensive legal review loop and front a simple Q&A surface. Useful case study in **cost-aware LLM batching** and **structured extraction** under real document noise.
